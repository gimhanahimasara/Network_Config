# Network_Config

Network configuration assignment

This document is an individual assignment for the Systems and Network Programming (IE2012) module at the Sri Lanka Institute of Information Technology (SLIIT). It serves as a comprehensive technical guide for setting up and managing a Linux-based environment, specifically using Ubuntu.

The assignment covers several key areas of systems administration and network programming:

1. Environment Setup
Virtualization: Detailed instructions for installing Oracle VirtualBox and configuring an Ubuntu 24.04 LTS virtual machine.

Linux Fundamentals: A summary of open-source Unix-like operating systems and a glossary of essential navigation and file manipulation commands (e.g., pwd, ls, mkdir, cp, mv).

2. Network Services Configuration
DHCP Server: Step-by-step guide to installing and configuring the isc-dhcp-server to automatically assign IP addresses within a network.

DNS Server: Procedures for setting up BIND9, including the configuration of forward and reverse lookup zones for domain name translation.

NTP (Network Time Protocol): Using the Chrony package to synchronize system clocks over a network.

SSH Server: Instructions for enabling secure remote login and command execution via OpenSSH.

3. Security and Automation
Firewalls and ACLs: Practical examples of using iptables to block social media sites and secure web browsing, as well as managing file-level permissions with Access Control Lists.

Shell Scripting: Introduction to automation using Bash scripts, including a specific example of an automated log cleanup process scheduled via cron jobs.

4. Application Servers
Web Server: Configuration of an Apache server to host a basic HTML webpage.

Email Server: Installation and testing of Postfix to manage digital mail communication.

5. Debugging and Analysis

Linux GDB: A section dedicated to software debugging using the GNU Debugger, illustrating how to set breakpoints, disassemble code, and inspect memory registers to analyze program behavior.
